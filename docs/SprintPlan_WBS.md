# Work Breakdown Structure (WBS) & Sprint Plan

## Agile Sprint Plan (2-Week Sprints)
**Sprint 1: Core Foundation & Identity**
- Setup solution architecture and Git repository.
- Build IdentityService (JWT, Login, Signup).
- Scaffold Angular shell and routing.

**Sprint 2: Catalog & Frontend Core**
- Build CatalogService (CRUD products, upload media).
- Implement PLP (Product Listing Page) in Angular.
- Setup Ocelot API Gateway routing for Auth and Catalog.

**Sprint 3: Workflow & Publishing**
- Build ProductWorkflowService (Pricing, Inventory, Status).
- Implement Publish Wizard (Stepper) in Angular.
- Connect Workflow API endpoints to Gateway.

**Sprint 4: Reporting, Polish & Testing**
- Build AdminReportingService (Audit Logs, Dashboards).
- Implement Storefront Preview and Dashboard UIs.
- End-to-End integration testing and bug fixes.
